## **SProcessInfo**

Struktura przechowująca informacje o podłączonych dyskach komputera

| Pole                                     | Opis                                     |
| ---------------------------------------- | ---------------------------------------- |
| `unsigned int PID`                       | ***Zawiera Process ID***                |
| `unsigned int SID`                       | ***Zawiera Session ID***                 |
| `std::string Name`                       | ***Zawiera nazwe procesu***              |
